MovieCube SG

Online Supermarket Website

Software: HTML, CSS, php, Bootstrap

Platform Used: NetBeans, MySQL

Date of Completion: 07/06/2023

Introduction to the Work

MovieCube SG is a PHP web application developed for the C203 Web Application Development course.

The project focuses on movie reviews and includes features such as user registration, login, movie display, and reviews.

Learning Points

This project provided hands-on experience in PHP, HTML, Bootstrap, CSS, and MySQL database integration. Key learning points include user authentication, database operations, and creating interactive web pages.

Key Features

  • User Registration and Login
  • Display List of Movies
  • Movie Reviews with User Interaction
  • Advanced Features like account updates, deletion, and review management
  • Database Integration with MySQL

Database Schema

The project utilizes a MySQL database with tables for users, movies, and reviews. Fields include user information, movie details, and review data.

The application focuses on delivering an intuitive and responsive user experience while adhering to the project requirements.

Key Features and Techniques

MovieCube SG highlights the effective use of HTML, PHP, Bootstrap, and CSS to create an interactive and dynamic web application. Key features and techniques include user registration and login functionality, a movie display page with advanced sorting options, and a comprehensive movie review system.

The project integrates Bootstrap for a responsive and visually appealing user interface. Elements such as forms, navigation bars, and grids were strategically employed to enhance the overall user experience and streamline the presentation of movie information.

Final Thoughts...

Reflecting on the MovieCube SG project, the incorporation of Bootstrap elements not only led to the development of a feature-rich movie review platform but also expanded my proficiency in web development. This project represents a culmination of learning, problem-solving, and successful implementation of technologies for an engaging and functional web application.
